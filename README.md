# Token Formations

**Token Formations** is a Foundry VTT module that enables coordinated movement and formation management for tokens on the map.

https://github.com/user-attachments/assets/ae9cfaf3-413d-4982-a1e9-39c07fcbd0b8

## Features

- **Formation Management:** Assign tokens to follow a leader, maintaining a configurable formation width and respecting maximum distance constraints.
- **Automatic Movement:** Followers move automatically behind their leader, avoiding obstacles and walls.
- **Visual Indicators:** Followers display a customizable indicator showing their leader's portrait and position.
- **Combat Integration:** Formations are disabled during combat and restored when combat ends.
- **Customizable Settings:** Adjust formation width, maximum distance, indicator appearance, and position.
- **Localization:** Supports English and Italian.

## Usage

- **Add a Follower:** Select a token, hover over the desired leader, and press `F`.
- **Remove a Follower:** Move the token manually or delete the leader.
- **Clear All Formations:** Use the "Clear Formations" button in the scene controls.
- **Adjust Settings:** Change options in the module settings panel.
